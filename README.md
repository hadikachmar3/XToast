XToast
A simple custom toast(Simplified version of SuperToasts).

**It supports the following configurations:**
 - Text size/color
 - 	Background color/resource
 - 	Animation
 - 	Gravity and offset
 - 	Cover previous toast
 - 	Integrate with a button(text/event)

All configurations are in the form of withXxx() methods, for example:
`XToast.create(Context, "Text to 
show").withTextSize(20).withDuration(2000).show()`
